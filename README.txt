# World Of Games
Welcome to the World Of Games application! Here you will find three exciting games to play.

## Memory Game
In Memory Game, a sequence of numbers will appear for 1 second and you have to guess it back. The goal is to remember the sequence and repeat it correctly.

## Guess Game
In Guess Game, you have to guess a number and see if you chose like the computer. The computer will randomly generate a number, and you have to guess it. If your guess is correct, you win!

## Currency Roulette
In Currency Roulette, you have to try and guess the value of a random amount of USD in ILS. You will be shown a random amount in USD and have to guess the equivalent value in ILS.

Have fun playing the World Of Games application!
